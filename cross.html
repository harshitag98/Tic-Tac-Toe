Welcome to Tic Tac Toe !!

Hey buddy, Welcome to the Tic Tac Toe game. If you are here to play Tic Tac Toe, congrats you are at the right place.

First let's go through the rules of the game.

  1. If you want to input 'X' in any cell of the board, simply click on that cell.
  2. If you want to input 'O' in any cell of the board, simply click on that cell.
  3. If any input character matches in three cells either horizontlly, vertically or diagonally, the user with that input character will win the game.
  4. If none of the users win the game, it will be a draw between the two.

Now that you have gone through the rules, you are ready to play the game.
